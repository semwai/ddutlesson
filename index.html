PHP

  1. Клиент-серверное взаимодействие. Сервер и клиент. Url, доменное имя, порт
  2. Установка и настройка серверного ПО (openserver)
  3. Основы синтаксиса PHP, комментарии, подключение на страницу, переменные и вывод информации на экран
  4. Типы данных, проверка на существование, удаление, проверка типа, константы
  5. Операторы (арифметические, логические, побитовые). Приоритет операторов. Тернарный оператор
  6. Условные конструкции, циклы, массивы
  7. Функции, аргуметы по умолчанию, переменное число аргументов
  8. Область видимости переменных
  9. Расширенная работа с массивами
  10. Работа со строками. Интерполяция переменных
  11. Подключение внешних файлов, пример разбиения сайта на компоненты
  12. Формы. Обработка форм, post\get, параметры, параметр-массив
  13. Формы и файлы. Работа с файлами
  14. Cookie. Разработка сайта-счетчика.
  15. Cookie. Пример простейшей авторизации (без регистрации)
  16. Классы, поля и методы, конструктор

  4. Простейшие запросы на вставку, удаление, изменение и чтение даннх (CRUD)

  6. Фильтрация запросов
  7. Объединение таблиц. Декартово произведение, левое\правое соединение
  8. Сложные\вложенные запросы
  9. Foreign Key
  10. Изменение таблиц
  11. Работа с дампами

MySQL + PHP

  1. Соединение с базой данных,
  2. Select .. + цикл

  4. PHP PDO
